# 🚑 Ambulance Detection System using YOLOv8

## 📌 Overview
This project detects ambulances from multiple video feeds using the YOLOv8 object detection model combined with color-based filtering. It identifies vehicles (trucks) and determines if they are ambulances based on red color dominance. A simulated traffic light system reacts in real-time.

---

## ✨ Features
- Real-time object detection using YOLOv8  
- Multi-video input support  
- Red color-based ambulance detection  
- Traffic light simulation (Red/Yellow/Green)  
- Side-by-side video display  

---

## 🛠️ Tech Stack
- Language: Python  
- Libraries: OpenCV, NumPy, Ultralytics YOLOv8
